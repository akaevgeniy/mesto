**Описание проекта**

Проект Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки. На сайте представлены фотографии с названиями посещенных мест. На данный момент реализована функции редактирования профиля, добавление карточек через template, увеличение рисунка, удаление карточек и лайк карточки.

**Использованные технологии**

При разработке сайта использованы технологии разрабтки адаптивных сайтов, принцип desctop first. Использован скриптовый язык JavaScript для реализации диалогового окна типа pop-up, template и объекты DOM. Сайт построен на основе макета Figma.
